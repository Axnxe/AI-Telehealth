# 🩺 Obas Lab AI Clinical Workflow Assistant

AI-powered prototype that assists with clinical documentation workflows by generating SOAP notes from visit summaries using synthetic patient data.

## 🚀 Features
- AI-generated SOAP notes (Groq LLM)
- Synthetic patient system (HIPAA-safe)
- Medication workflow simulation (no real prescriptions)
- Clinician review before saving output
- Streamlit web interface

## ⚙️ Tech Stack
- Python
- Streamlit
- Groq API (LLaMA 3)
- Faker (synthetic data)
- dotenv

## 🧠 Purpose
Designed to reduce repetitive clinical documentation workload by assisting with SOAP note drafting during patient visits.

## ⚠️ Disclaimer
This is a prototype only. No real patient data is stored or processed.
