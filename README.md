# 🩺 Obas Lab AI Telehealth Assistant

AI-powered clinical workflow prototype that generates SOAP notes from patient visit summaries using Groq LLMs and synthetic patient data.

## Features
- AI SOAP note generation (Groq LLM)
- Synthetic patient system (HIPAA-safe)
- Medication workflow simulation
- Clinician review step
- Streamlit web interface

## Disclaimer
This is a prototype only. No real patient data is stored or processed.
